The backlog_overview.ipynb Jupyter Notebook contains code that was used to generate the data that informed the backlog portions of the dashboard pictured above, which was included in VA's Cleland-Dole Act Report. The .xlsx and .csv files were used in the code to create the backlog_overview.xlsx file, which was fed into Power BI to generate the dashboard. These files were sources from foia.gov, a dashboard used for the previous year's report, and a file generated by the code in the "VA Volume, Number Processed and Received.ipynb" Jupyter notebook.
